# Physics-Informed Neural Networks (PINNs) for Dynamic Systems

This repository contains a comprehensive tutorial and implementation of a **Physics-Informed Neural Network (PINN)** applied to the **Damped Harmonic Oscillator**. 

## 🎯 Project Objective
Traditional Neural Networks are often "black boxes" that ignore physical constraints. This project demonstrates how to embed Ordinary Differential Equations (ODEs) into the loss function of a Deep Learning model using PyTorch's Automatic Differentiation.

## 🚀 Key Features
* **Physics-Driven Loss**: Implements a custom loss function that penalizes violations of Newton's Second Law.
* **Data Efficiency**: Shows how PINNs can converge with 80% less training data than standard MLPs.
* **Accessibility-First**: All visualizations use the `Okabe-Ito` color palette and distinct line styles for color-blind accessibility.
